# IPv6 Configuration Commands

## Enable IPv6 Routing

| Command | Description |
|---------|-------------|
| `ipv6 unicast-routing` | Enable IPv6 routing globally |

## IPv6 Address Configuration

| Command | Description |
|---------|-------------|
| `ipv6 address <address>/<prefix>` | Assign IPv6 address to interface |
| `ipv6 address <address>/<prefix> eui-64` | Use EUI-64 for interface ID |
| `ipv6 address <address> link-local` | Configure link-local address |
| `ipv6 address autoconfig` | Enable SLAAC |
| `ipv6 address dhcp` | Obtain address via DHCPv6 |
| `ipv6 enable` | Enable IPv6 and generate link-local |

## IPv6 Address Examples

```
! Manual configuration
interface g0/0
 ipv6 address 2001:db8:1::1/64
 no shutdown

! EUI-64
interface g0/1
 ipv6 address 2001:db8:2::/64 eui-64
 no shutdown

! Link-local only
interface g0/2
 ipv6 enable
 no shutdown
```

## IPv6 Static Routing

| Command | Description |
|---------|-------------|
| `ipv6 route <prefix>/<length> <next-hop>` | Static route via next-hop |
| `ipv6 route <prefix>/<length> <interface>` | Static route via interface |
| `ipv6 route <prefix>/<length> <interface> <next-hop>` | Fully specified route |
| `ipv6 route ::/0 <next-hop>` | Default route |

## IPv6 Neighbor Discovery

| Command | Description |
|---------|-------------|
| `show ipv6 neighbors` | Display IPv6 neighbor table (like ARP) |
| `clear ipv6 neighbors` | Clear neighbor cache |

## IPv6 Show Commands

| Command | Description |
|---------|-------------|
| `show ipv6 interface` | Display IPv6 interface details |
| `show ipv6 interface brief` | Display IPv6 interface summary |
| `show ipv6 route` | Display IPv6 routing table |
| `show ipv6 route static` | Display IPv6 static routes |
| `show ipv6 route connected` | Display connected routes |
| `show ipv6 route <protocol>` | Display routes from protocol |
| `show ipv6 protocols` | Display IPv6 routing protocols |

## IPv6 Address Types

| Type | Prefix | Description |
|------|--------|-------------|
| **Global Unicast** | 2000::/3 | Routable on Internet |
| **Link-Local** | FE80::/10 | Local link only |
| **Unique Local** | FC00::/7 | Private addresses |
| **Multicast** | FF00::/8 | One-to-many |
| **Loopback** | ::1/128 | Localhost |
| **Unspecified** | ::/128 | No address |

## Common Multicast Addresses

| Address | Description |
|---------|-------------|
| `FF02::1` | All nodes (link-local) |
| `FF02::2` | All routers (link-local) |
| `FF02::5` | OSPF routers |
| `FF02::6` | OSPF DRs |
| `FF02::9` | RIPng routers |
| `FF02::A` | EIGRP routers |
| `FF02::1:FF__:____` | Solicited-node multicast |

## DHCPv6

### Stateless DHCPv6 (SLAAC + DHCPv6)

```
! Router config
ipv6 dhcp pool STATELESS
 dns-server 2001:db8::53
 domain-name example.com

interface g0/0
 ipv6 nd other-config-flag
 ipv6 dhcp server STATELESS
```

### Stateful DHCPv6

```
! Router config
ipv6 dhcp pool STATEFUL
 address prefix 2001:db8:1::/64
 dns-server 2001:db8::53
 domain-name example.com

interface g0/0
 ipv6 nd managed-config-flag
 ipv6 nd prefix default no-autoconfig
 ipv6 dhcp server STATEFUL
```

## IPv6 Tunneling

| Command | Description |
|---------|-------------|
| `tunnel mode ipv6ip` | Configure 6to4 tunnel |
| `tunnel source <interface>` | Set tunnel source |
| `tunnel destination <ipv4-address>` | Set tunnel destination |

## EUI-64 Process

1. Take 48-bit MAC address
2. Split in half: OUI | Device ID
3. Insert FFFE in middle
4. Flip 7th bit (U/L bit)
5. Result: 64-bit Interface ID
